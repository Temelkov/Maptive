Check out Some More Maptive Features

Color Markers by Category

Change the color of your marker based on categories within your data. Select a column in your spreadseet and each unique value will have its own colored marker.

Draw Radius on Map

Click on a location or enter an address to add a specific raius around your map object. Then click on the radius to export all of your locations inside the circle!

Export a Map Image

Take an Image of the map in its current state to use in a PowerPoint presentation, or take a large image and print it out to hang on the wall.

Filter Tool

Precission level control makes it easy to filter data so that you only see what you want to on the map you create and customize.

Route Optimization / Directions

Map the most efficient route between multiple locations and, whit the trusty ald of our Proximity tool, show nearby points from any specified location.

Create Sales Territories

Draw yor own sales territories, or create them using predefined boundaries like zip codes. Sales Territories can aggregate important data, like total sales and even demographic data, for all the makers in them.
